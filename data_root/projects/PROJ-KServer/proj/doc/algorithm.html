The k-server algorithm returns a string indicating the indexes of the servers that are required to service the incoming requests.

Input: Two two dimensional arrays of integers representing servers, requests and their respective locations.
Output: A string of integers seperated by commas.
Test of correctness:     If there are as many valid integers as there are requests the output is declared correct.
Implementation: To implement a k-server algorithm the author has to write a java class extending the KServerAbsAlgorithm class
